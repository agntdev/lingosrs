# LingoSRS

A private Telegram bot offering spaced-repetition flashcards for language learners—create/import decks, run quick reviews, and track progress with configurable reminders.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
